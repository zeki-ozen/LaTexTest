# LaTexTest
# Test Commiti
A simple LaTeX test repository demonstrating basic LaTeX functionality.

## Contents

- `test.tex` - A basic LaTeX document with examples of:
  - Document structure
  - Mathematical expressions (inline and display)
  - Lists (itemized and enumerated)
  - Sections and subsections

## Compiling the Test Document

To compile the LaTeX document, use:

```bash
pdflatex test.tex
```

Or use any LaTeX distribution (TeX Live, MiKTeX, etc.) to compile the document.

## Requirements

- A LaTeX distribution (TeX Live, MiKTeX, MacTeX, etc.)
- The following packages (usually included in standard distributions):
  - inputenc
  - amsmath

## Output

The compilation will generate a PDF file (`test.pdf`) containing formatted text and mathematical expressions.
